# iView-project

This project is build for Vue.js 2 + vue-router + webpack2 + iView 2, just install and run.

## Install
```bush
// install dependencies
npm install
```

##注意 install时 添加执行参数 uuid --save 否则下面init时又编译不过

## Run
### Development
```bush
// For the first time, run init to create index.html
npm run init
npm run dev
```
### Production(Build)
```bush
npm run build
```

### 初始化开发环境
### dev是普通的开发环境
